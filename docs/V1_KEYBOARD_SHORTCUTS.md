# V1 Keyboard Shortcuts (Reference)

## Global (Dashboard)
- `/` Focus Quick Add
- `t` Open Timeline page
- `c` Edit Countdown (open modal)
- `?` Keyboard help overlay
- `Esc` Close modal/overlay (and blur inputs)

## Tasks list
- `↑` / `↓` Move selection
- `Space` Toggle completion for selected task
- `Enter` Add task when Quick Add focused
- `Backspace` Delete selected task (with Undo toast)
- `u` Undo last destructive action (best-effort V1)

## Timeline page
- `Esc` Close any open editor panel/popover
- `n` New project (optional)
- `b` Back to dashboard (optional)

Notes:
- If a shortcut conflicts with typing in an input, shortcuts should be disabled while an input is focused (except `Esc`).
